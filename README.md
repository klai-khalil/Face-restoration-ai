\# AI Face Restoration for Surveillance Videos



\## Overview



This project focuses on improving facial quality in low-quality surveillance videos using deep learning techniques.



The proposed pipeline combines:



\- Image denoising

\- Super-resolution

\- Face restoration



\## Objectives



\- Improve facial details

\- Enhance low-resolution faces

\- Evaluate restoration quality



\## Technologies



\- Python

\- PyTorch

\- OpenCV

\- Real-ESRGAN

\- GFPGAN
